
# 🌍 EpiGuard AI
WHO-style epidemic intelligence command center for outbreak prediction, hotspot detection, and risk surveillance.

## 🚀 Features
- pseudo real-time outbreak ingestion
- ML forecasting with Prophet
- surge alert detection
- WHO-style risk heatmap
- Streamlit dashboard
- country-level risk intelligence

## 📊 Datasets
- Johns Hopkins COVID-19 dataset
- OWID secondary epidemiology data
- optional mobility enrichment

## 🤖 ML Models
- Prophet
- XGBoost baseline
- risk scoring logic
